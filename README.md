Бонусный кошелек КРЕПКО
Мобильное приложение для управления программой лояльности магазина «КРЕПКО» (розничная продажа крепежных изделий, метизов и инструмента).

О приложении
Приложение предназначено для автоматизации работы с бонусной системой магазина и разделено на две роли:

Режим Пользователь (клиент магазина)

Авторизация по логину и паролю

Просмотр текущего баланса бонусных баллов

История покупок с начисленными и списанными бонусами

Отображение QR-кода (виртуальная карта лояльности)

Режим Администратор (сотрудник магазина)

Авторизация по логину и паролю

Просмотр списка всех зарегистрированных пользователей

Редактирование данных пользователей (ФИО, телефон, email)

Ручная корректировка баланса бонусов

Добавление новых пользователей

Технологии
Язык: Kotlin

Среда разработки: Android Studio Narwhal 4 (2025.1.4)

Минимальная версия Android: API 24 (Android 7.0)

Хранение данных: JSON-файл во внутреннем хранилище

Работа с JSON: Gson

Особенности реализации
Авторизация: проверка логина и пароля по данным из JSON-файла

Роли: разделение функционала через поле role в модели User

Хранение данных: все пользователи хранятся в файле users.json во внутреннем хранилище приложения

Персистентность: данные сохраняются между сеансами работы
